chap1linux是什么

  • 重点回顾
    • 计算机主要以二进制作为单位，而目前常用的磁盘容量单位为B，其单位换算为1B=8bit，其他单位以1024为其倍数
    • 操作系统(Operation System, OS)主要用于管理与驱动硬件，因此必须要能够管理内存、管理设备、负责进程管理以及系统调用等。因此，只要能够让硬件准备妥当(Ready)的情况，就是一个很棒的操作系统了
    • 操作系统重点仅在驱动与管理硬件，而要使用硬件时，就得需要通过应用软件或者是shell的功能来调用操作系统操纵硬件工作。因此，目前操作系统除了上述功能外，通常已经通常已经包含了日常弓箭所需要的应用软件了
    • UNIX的前身是由贝尔实验室(Bell Lab.)的Ken Thompson利用汇编语言携程的，后来在1971~1973年间由Dennis Ritchie以C程序语言进行改写，才成为UNIX
    • 1977年由Bill Joy释出BSD(Berkeley Software Distribution)，这些成为UNIX like的操作系统
